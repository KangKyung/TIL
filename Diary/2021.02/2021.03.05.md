## **해결 방법**

> git stash

다른 branch로 넘어갈 때, stash를 통해 변경 내역을 임시보관소에 넣어둘 수 있어요!
<br>git stash로 임시보관소에 넣고,
<br>git stash list로 보관 내역을 확인해볼 수 있고,
<br> git stash apply [stash 이름] 으로 해당 stash로 돌릴 수 있어요!
